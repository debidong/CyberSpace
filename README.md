# Running server

1. In `/CyberSpace/Backend` run

    ```bash
    python manage.py runserver
    ```

2. In `/CyberSpace/Backend/Redis-x64-5.0.14.1` run

   ```bash
   .\redis-server.exe .\redis.windows.conf
   ```

3. In `/CyberSpace/Frontend/CyberSpace` run

   ```bash
   npm run build
   ```
